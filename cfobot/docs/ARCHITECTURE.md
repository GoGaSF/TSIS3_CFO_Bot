# CFO Bot Architecture

The application follows a monolithic architecture built with Spring Boot.

The system includes three main layers:

Controller Layer
Handles HTTP requests and exposes REST endpoints.

Service Layer
Contains business logic for cloud cost calculation.

Model Layer
Defines pricing constants and data structures.

The frontend is implemented as static HTML, CSS, and JavaScript files served by the Spring Boot application.

Architecture type:
Monolithic Web Application

Components:
- REST API
- Chatbot Interface
- Pricing Engine
- Swagger API Documentation

Advantages of this architecture:
- simple deployment
- minimal infrastructure
- easy demonstration
- easy maintenance
